
# SemiProject 요일, 요양일지    

---

## 프로젝트 소개
##### 코로나 시국에 재활요양병원 작업치료사로써 근무하면서 입원환자가 매일 치료를 받는데 보호자가 환자 치료를 경과를 인지하면서
##### 치료에 도움이 될 정보의 필요성을 느끼게 되어 개발하게 되었습니다.
